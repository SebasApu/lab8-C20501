# 🎬 TVMaze Series Finder

Proyecto modular para buscar series usando la API de TVMaze.

## 📁 Estructura del Proyecto

```
proyecto/
├── index.html              # Página principal
├── src/
│   ├── css/
│   │   └── styles.css      # Estilos de la aplicación
│   └── js/
│       ├── main.js         # Punto de entrada de la app
│       ├── components/     # Componentes UI reutilizables
│       │   ├── ShowCard.js      # Tarjetas de series
│       │   ├── EpisodeCard.js   # Tarjetas de episodios
│       │   ├── Modal.js         # Modal de episodios
│       │   └── LoadingStates.js # Estados de carga/error
│       └── modules/
│           ├── data.js     # Datos quemados (hardcoded)
│           ├── api.js      # Funciones para la API
│           └── ui.js       # Coordinador de UI
└── README.md
```

## 🚀 Características

- ✅ **API Real**: Siempre usa la API de TVMaze
- ✅ **Carga inicial**: Muestra series populares al arrancar
- ✅ **Búsqueda en tiempo real**: Busca cualquier serie
- ✅ **Vista de episodios estilo Netflix**: Modal oscuro con tarjetas clickeables
- ✅ **Ratings colorizados**: Sistema de colores según valoración (0-10)
- ✅ **Diseño responsive**: Se adapta a móviles y desktop
- ✅ **Arquitectura modular con componentes**: Código organizado y reutilizable
- ✅ **Fallback inteligente**: Si la API falla, usa datos quemados

## 📦 Instalación

1. Clona o descarga el proyecto
2. Abre el proyecto en tu editor favorito
3. Instala las dependencias:

```bash
pnpm install
```

4. Ejecuta el servidor:

```bash
pnpm dev
```

5. Abre tu navegador en `http://localhost:3000`

## 🎯 Cómo Usar

### Carga Inicial

Al abrir la aplicación, automáticamente se cargan series populares desde la API de TVMaze. 

### Búsqueda

1. Escribe el nombre de una serie en el buscador (ej: "Breaking Bad", "Friends", "The Office")
2. Presiona "Buscar" o Enter
3. Los resultados aparecerán en tiempo real desde la API

### Datos Quemados

Los datos quemados solo se usan como **respaldo** si:
- La API de TVMaze no está disponible
- Hay problemas de conexión a internet
- El servidor de TVMaze está caído

En condiciones normales, **siempre** se usa la API real.

## 🔧 Arquitectura Explicada

### Módulos

#### `data.js`
Contiene un array con datos quemados de series reales de TVMaze. Estos datos tienen la misma estructura que los que devuelve la API y **solo se usan como respaldo** si la API falla.

#### `api.js`
Funciones para interactuar con la API de TVMaze:
- `searchShows(query)`: Busca series por nombre
- `getShowById(id)`: Obtiene detalles de una serie específica
- `getShowEpisodes(id)`: Obtiene los episodios de una serie
- `getPopularShows()`: Obtiene series populares para la carga inicial

#### `ui.js`
Coordinador de UI que importa y re-exporta componentes:
- Orquesta la comunicación entre componentes
- Maneja el renderizado principal de series
- Re-exporta funciones de componentes para fácil acceso

### Componentes

#### `ShowCard.js`
- Renderiza tarjetas de series con su rating colorizado
- Aplica clases de rating según el valor (0-10)
- Maneja los event listeners de las tarjetas

#### `EpisodeCard.js`
- Crea tarjetas de episodios estilo Netflix
- Incluye imágenes, metadata y ratings colorizados
- Abre enlaces a TVMaze al hacer click

#### `Modal.js`
- Gestiona el modal de episodios
- Agrupa episodios por temporada
- Maneja estados de carga y error del modal

#### `LoadingStates.js`
- Componentes reutilizables para mostrar carga
- Mensajes de error
- Estados de "sin resultados"

### `main.js`
Punto de entrada de la aplicación:
- Inicializa la app y carga series desde la API
- Maneja la búsqueda en tiempo real
- Configura event listeners globales
- Usa datos quemados como fallback si la API falla
- Coordina todos los módulos y componentes

## 📚 API de TVMaze

### Endpoints Usados

```javascript
// Buscar series
GET https://api.tvmaze.com/search/shows?q=breaking+bad

// Obtener serie por ID
GET https://api.tvmaze.com/shows/169

// Listar series
GET https://api.tvmaze.com/shows
```

### Documentación Completa
👉 https://www.tvmaze.com/api

## 🎨 Personalización

### Cambiar los Colores

Edita las variables CSS en `styles.css`:

```css
:root {
    --primary: #ff6b6b;     /* Color principal */
    --secondary: #4ecdc4;   /* Color secundario */
    --dark: #2d3436;        /* Color de texto */
    /* ... más colores */
}
```

### Agregar Más Datos Quemados

Edita el archivo `src/js/modules/data.js` y agrega más series al array `hardcodedShows`.

### Modificar el Layout

El grid se ajusta automáticamente, pero puedes cambiar el tamaño mínimo de las tarjetas en `styles.css`:

```css
.results-grid {
    grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
    /* Cambia 280px por el tamaño que prefieras */
}
```

## 💡 Tips para Desarrollo

- La aplicación **siempre usa la API real** de TVMaze
- Los datos quemados son un **respaldo automático** si falla la conexión
- Revisa la consola del navegador para ver logs útiles
- La API de TVMaze es gratuita y no requiere API key
- Las imágenes se cargan con `loading="lazy"` para mejor performance
- La carga inicial hace varias búsquedas para mostrar series variadas

## 🐛 Troubleshooting

**No se ven las series al cargar:**
- Verifica tu conexión a internet
- Revisa la consola del navegador por errores
- Si la API falla, deberías ver automáticamente los datos quemados

**No aparecen resultados al buscar:**
- Verifica la consola del navegador
- Asegúrate de escribir un término de búsqueda válido
- Intenta con nombres de series populares primero

**Los estilos no se aplican:**
- Verifica que la ruta en el `<link>` del HTML sea correcta
- Refresca la página con Ctrl+F5 (hard refresh)

**Los módulos no cargan:**
- Asegúrate de estar usando un servidor (como `servor`)
- Los módulos ES6 requieren un servidor, no funcionan con `file://`
